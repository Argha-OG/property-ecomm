# How to Use AI Search (Demo)

1.  **Select the "AI Search" tab** in the Hero section.
2.  **Type a natural language query**, for example:
    *   "Looking for a 3 bedroom condo in Mont Kiara under 2 million"
    *   "I want a bungalow in Damansara with 5 bedrooms"
    *   "Shop lot in Petaling Jaya for business"
3.  **Press Search**.
4.  The system will parse your query and redirect you to the listing page with the appropriate filters applied.

**Note:** This is a demonstration feature using keyword matching. It supports:
*   **Locations:** Kuala Lumpur, Selangor, Penang, Johor, Mont Kiara, Bangsar, Petaling Jaya, etc.
*   **Types:** Condo, Apartment, Bungalow, Commercial, Land.
*   **Price:** "under X million", "below X k", "max X m".
*   **Bedrooms:** "3 bed", "4 rooms", etc.
